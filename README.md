# ⏰ AI-Based Alarm Suggestion System

An intelligent alarm clock web application that sets alarms and suggests optimal wake-up times using simple AI-based logic.

---

## 🚀 Features
- 🕒 Set alarm (AM/PM format)
- 🤖 AI-based wake-up time suggestion
- 🔊 Alarm sound notification (alarm.mp3)
- 🌙 Light/Dark theme toggle
- 📱 Simple and responsive UI
- ⏳ Real-time alarm checking system

---

## 🧠 AI Logic (Simple)
The system suggests wake-up time based on user input sleep duration and current time.

Example:
If sleep time = 6 hours → system suggests wake-up time = current time + 6 hours

## ⚙️ How to Run
1. Open project folder  
2. Ensure `alarm.mp3` is in same directory  
3. Open `index.html` in browser  
4. Set alarm time  
5. Click Start button  
6. Alarm will ring at set time 🔔  

---

## 🔊 Important Notes
- Browser requires user interaction to play sound  
- File name must be exactly `alarm.mp3`  
- Keep volume enabled
